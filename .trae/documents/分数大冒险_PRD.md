# 分数大冒险 · 儿童分数启蒙游戏 PRD

## 1. 产品概述
- 一款面向 6-10 岁儿童的桌面端（浏览器运行）分数启蒙游戏，让孩子通过切披萨、数轴跳跃、拼图合并等可视化操作，理解等分、约分与简单分数加减。游戏不强调竞速与排名，重在每个知识点"看见、摸到、做对"。
- 目标价值：用厨房与拼图的具象情境替代抽象符号，配合语音引导、错误重试与星级鼓励，降低分数入门门槛；同时为家长提供掌握度、常错题型与游戏时长的可视化报告。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 小玩家 | 直接进入关卡选择 | 游玩五大关卡、领取奖励、回看收藏 |
| 家长 | 关卡选择页"家长入口"（可设简易口令） | 查看掌握知识点、常错题型、游戏时长，重置进度 |

### 2.2 功能模块
1. **关卡选择**：三大主题岛屿 + 总览进度，家长入口与奖励收藏入口。
2. **厨房切分**：把披萨切成等份，识别每块代表几分之几。
3. **数轴跳跃**：把分数卡拖到数轴正确位置，理解分数大小。
4. **拼图挑战**：约分合并、比较大小、补全分子、简单加减四类小关卡。
5. **奖励收藏**：徽章贴纸册 + 星星总览 + 知识点小图鉴。
6. **家长面板**：掌握知识点清单、常错题型排行、游戏时长统计。

### 2.3 页面详情
| 页面 | 模块 | 功能描述 |
|------|------|----------|
| 关卡选择 | 主题岛屿地图 | 三大主题岛（厨房/数轴/拼图），每岛显示关卡节点与已得星星；底部奖励册与家长入口按钮 |
| 关卡选择 | 顶部状态条 | 玩家昵称、累计星星、当前难度等级、声音开关 |
| 厨房切分 | 披萨操作台 | 圆形披萨，点击"切 N 刀"按钮切出等份；语音读题"把披萨平均分成 4 份" |
| 厨房切分 | 分数作答 | 切完后选择/拖出每块是几分之几，答对进入下一题，答错可重试 |
| 数轴跳跃 | 数轴场景 | 0-1（或 0-2）数轴带刻度标记，下方为待放置的分数卡 |
| 数轴跳跃 | 拖拽放置 | 拖动分数卡到数轴位置，靠近正确点自动吸附并播放鼓励语音 |
| 拼图挑战 | 约分合并 | 多张同分母卡（如 2/4）可合并为约分卡（1/2），拖动合并 |
| 拼图挑战 | 比较大小 | 两张分数卡，点选较大/较小的一方，配天平动画 |
| 拼图挑战 | 补全分子 | 给出等式 1/? = 2/6，拖入正确分子数字块 |
| 拼图挑战 | 简单加减 | 同分母分数加法/减法，用披萨图辅助表示结果 |
| 奖励收藏 | 徽章贴纸册 | 已解锁徽章亮起，未解锁灰色提示解锁条件 |
| 奖励收藏 | 星星总览 | 每关三星进度条 + 总星星数 |
| 奖励收藏 | 知识点图鉴 | 已掌握知识点以可爱卡片呈现，点击可语音复述定义 |
| 家长面板 | 掌握知识点 | 知识点清单与掌握状态（未接触/练习中/已掌握） |
| 家长面板 | 常错题型 | 按错误次数排序的题型列表，含错误次数与正确率 |
| 家长面板 | 游戏时长 | 总时长、各主题时长、本周累计柱状图 |

## 3. 核心流程

玩家从关卡选择进入某一主题关卡 → 语音读题 → 进行可视化操作作答 → 答对得星星并解锁知识点、答错可重试不扣分 → 完成整关结算星级（按首次答对率）→ 返回关卡选择，进度与奖励实时入册 → 家长可随时进入家长面板查看报告。

## 4. 用户界面设计

### 4.1 设计风格
- **设计方向**：温暖的"绘本厨房"风格，像翻开一本立体纸艺绘本，圆润、手作、带轻微倾斜的贴纸感，避免冷峻的扁平科技风。
- **主色**：奶油底 `#FFF6E5`、番茄红 `#E85A4F`、奶酪黄 `#F6B73C`、罗勒绿 `#7CB342`、面团棕 `#8B6F47`、天空蓝 `#5B9BD5`。
- **辅助**：深棕文字 `#3E2C23`、米白卡 `#FFFDF7`、阴影柔光 `rgba(139,111,71,0.18)`。
- **按钮**：圆角胶囊/圆角矩形，带内高光与底部柔阴影的"贴纸"按钮，按下时轻微下沉。
- **字体**：标题用圆润手写感字体 Fredoka / Baloo 2；中文标题用站酷快乐体或思源黑体加粗；正文用思源黑体 / Noto Sans SC。
- **布局**：桌面优先居中卡片式，大量留白，关卡选择为岛屿地图横向滚动。
- **图标/emoji**：🍕🍕🍕🍕🍕 披萨、⚖️ 天平、⭐ 星星、🏅 徽章，搭配自绘 SVG 食材。
- **动效**：页面切换淡入+轻微上浮；答对时星星迸发+披萨块弹跳；答错时温柔摇晃+鼓励语音，绝不出现红色叉号。

### 4.2 页面设计概览
| 页面 | 模块 | UI 元素 |
|------|------|--------|
| 关卡选择 | 主题岛屿地图 | 奶油渐变背景+纸纹，三座弧形岛屿，岛上有圆角关卡节点（带星星），节点用虚线小路相连 |
| 关卡选择 | 顶部状态条 | 玩家头像、昵称、星星计数、难度标签、喇叭开关 |
| 厨房切分 | 披萨操作台 | 木质砧板背景，中心圆形披萨（番茄底+奶酪+配菜），下方"切 2 刀/切 3 刀"贴纸按钮，题目气泡 |
| 厨房切分 | 分数作答区 | 切完后弹出分数选项卡，选中高亮，作答正确披萨块发光 |
| 数轴跳跃 | 数轴场景 | 横向木条数轴，刻度用小钉子，0/1 端点为木桩，待放分数卡在下方圆角卡座 |
| 数轴跳跃 | 拖拽放置 | 卡片拖动有投影与轻微旋转，吸附时数轴对应位置闪金光 |
| 拼图挑战 | 约分合并 | 同分母卡同色系，合并时两张卡飞向中央融合为一张约分卡 |
| 拼图挑战 | 比较大小 | 天平横梁动画，较重一侧下沉，分数卡分别置于两端托盘 |
| 拼图挑战 | 补全分子 | 等式横排，缺失分子位置为虚线凹槽，数字块从下方库拖入 |
| 拼图挑战 | 简单加减 | 两块披萨图叠加/拿走，结果区显示合成披萨 |
| 奖励收藏 | 徽章贴纸册 | 翻页式贴纸册，徽章为圆形彩色印章风，未解锁为剪影 |
| 奖励收藏 | 星星总览 | 每关三颗星槽位，已得星发光，总星星大数字徽章 |
| 家长面板 | 掌握知识点 | 清单表格：知识点/状态/练习次数/正确率，状态用色块标签 |
| 家长面板 | 常错题型 | 横向条形图，红色条代表错误次数，悬停看正确率 |
| 家长面板 | 游戏时长 | 总时长大数字 + 各主题环形占比 + 本周柱状图 |

### 4.3 响应式
- 桌面优先（最小宽度 1024px 居中布局），平板（768-1024px）自适应缩放，触摸交互优化（拖拽、点击区域 ≥ 48px）。不做手机竖屏适配（桌面游戏定位）。

### 4.4 无障碍与儿童友好
- 所有交互元素 ≥ 48px，色彩对比度 ≥ AA；语音提示可重复播放；错误反馈温和无否定词；进度自动保存，关闭再开可续玩。
